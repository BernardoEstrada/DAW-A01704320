Conceptos

URL

(Uniform Resource Locator) Es una serie de caracteres que definen la ubicacion de un recurso en una red o el internet

MIME

(Multipurpose Internet Mail Extension) Es un estandar de formato para emails que permite el uso de caracteres especiales y archivos adjuntos.

HTTP

Existen 9 metodos HTTP:
GET: Solicita un recurso especificado
HEAD: Misma funcion que GET pero sin el body de la respuesta
POST: Envia una entidad al recurso especificado
PUT: Misma funcion que POST pero reemplaza la entidad existente
DELETE: Elimina el recurso especificado
CONNECT: Establece una coneccion al servidor
OPTIONS: Define las opcioines de comunicación
TRACE: Genera un mensaje loop-back en el camino al recurso
PATCH: Aplica modificaciones parciales a un recurso
Y 5 tipos de errores:
Respuestas Informales: 100-199
Respuestas Exitosas: 200-299
Redirecciones: 300-399
Errores de Cliente: 400-499
Errores de Servidor: 500-599
Mas informacion sobre los errores HTTP aquí

HTML5

(Hypertext Markup Language 5) Es la ultima versión de el estandar de hipertexto con el que se permite hacer paginas web con elementos mas poderosos y mejor organizados

JS

JavaScript es un lenguaje de programación interpretado con el que se puede agregar funcionalidad a paginas y aplicaciones web, así como a servidores.

PHP

Es un lenguaje de secuencias usado principalmente para desarrollo web

JSP

JavaServer Pages es una tecnologia que simplifica la creacion de contenido web dinamico.

AJAX

Es una herramienta que permite modificar la página sin la necesitar de actualizarla, recibir información del servidor despues de que la pagina haya terminado de cargar y enviar informacion al servidor en el fondo
