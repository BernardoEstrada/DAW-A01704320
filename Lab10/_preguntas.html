  • $_​COOKIE - Cookies de http
  • $_​SESSION - Variables de sesión
  • $_​REQUEST - Variables de un Request en http

¿Qué es XSS y cómo se puede prevenir?

XSS (Cross-site scripting) es una manera de inyectar código (posiblemente malicioso) a una página con fines de modificar su funcionamiento. Se puede prevenir tomando mediadas especiales al procesar datos que el usuario ingresa.
